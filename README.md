# Seaborn
Seaborn is a library for making statistical graphics in Python. It is built on top of matplotlib and closely integrated with pandas data structures.

## Installation

> pip install seaborn

## Dependencies

> Python 3.6+

## Mandatory Dependencies

> numpy (>= 1.13.3)

> scipy (>= 1.0.1)

> pandas (>= 0.22.0)

> matplotlib (>= 2.1.2)
